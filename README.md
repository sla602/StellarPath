# StellarPath

AI-powered career visualization platform for Women.

Inspiration
Woman often struggle to understand how their current skills connect to their future careers in a new country. Resumes, skills, and networking can feel scattered and hard to visualize. We wanted to create a tool that makes career growth clearer and more engaging by turning it into something people can explore visually

What it does
StellarPath is an AI-powered platform that visualizes a user's skills, experiences, and professional connections as a galaxy. By analyzing a resume and target role, the system maps skills as constellations and shows how they relate to career goals. Users can explore their personal skill universe and discover connections that may help them move toward their desired career.

How we built it
We built StellarPath using React and Vite for the frontend and TailwindCSS and Framer Motion for smooth UI interactions. We integrated the Gemini API to analyze user input such as resumes, experiences, and target jobs. The AI extracts relevant skills and generates insights that populate the constellation map.

Challenges we ran into
One of the biggest challenges was translating abstract career data into a visual structure that feels intuitive. Mapping skills and relationships into a graph while keeping the interface readable required careful tuning. Another challenge was integrating AI responses into the visualization pipeline so that the generated insights could dynamically update the graph structure.

Accomplishments that we're proud of
We are proud that we successfully turned career development into an interactive visual experience. The galaxy metaphor makes complex information easier and more engaging to explore.

We also managed to combine AI analysis with real-time data visualization, allowing users to immediately see their skills represented in a meaningful way.

What we learned
Through this project we learned how to combine AI-generated insights with interactive data visualization. We also gained experience working with graph-based UI libraries and designing systems that transform unstructured text into structured visual data.

What's next for StellarPath
Next, we want to improve the accuracy of skill extraction and add personalized recommendations for career growth. We also plan to expand the networking feature so users can discover potential collaborators and mentors within their constellation network.

In the future, StellarPath could evolve into a platform that helps students continuously map and grow their professional universe.
